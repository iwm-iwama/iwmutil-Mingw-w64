### これは何？
* MinGW-w64 `GCC版`による、**C/C++ライブラリ**です。
* 関連サイト
	- [MSYS2](https://www.msys2.org/)

---

### 特徴は？
* Windows 64bit版のコマンドライン・プログラム作成を支援します。
* 動的メモリ管理
	* **手動が基本**
		- ヒープ領域の割り当て／解放は、`icalloc()` `irealloc()` `ifree()`と、その派生関数／マクロで行います。
	* **終了時の自動解放**
		- 終了時点で解放されていないメモリを自動解放します。
		- 強制終了`[Ctrl]+[C]`も同様です。
	* **セキュリティ**
		- 割り当て／解放時にメモリを**ゼロクリア**します。
* 日付計算
	* ユリウス暦`-4712/01/01..1582/10/04`と、グレゴリオ暦`1582/10/15..9999/12/31`に対応。
	* 存在しない`1582/10/05..1582/10/14`を考慮。
	* `数ヶ月前後`の計算に対応。
		- `05/31`の`1ヶ月後`は`06/30`
		- `06/30`の`1ヶ月前`は`05/30`
* 標準入力
	- `パイプ経由`と`引数`のデータの有無を判定する関数を用意。
* UTF-8／CP65001
	- 基本実装は`UTF-8`。
	- ただし、`Shift_JIS／CP932を判定` `Shift_JISとUTF-8を交互に変換`する関数を用意。
